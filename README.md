# PDAC_CSC
ARPC1B+ cancer stem cells as mediators of gemcitabine resistance in pancreatic cancer. Targeting ARPC1B with CK-636 sensitized tumors to gemcitabine and inhibited growth, highlighting ARPC1B+ CSCs as a potential therapeutic target.
